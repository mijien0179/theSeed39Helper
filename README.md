# theSeed39Searcher
메이플스토리 더시드 39층 문제 검색기입니다.

* 창을 맨 위로 유지시킬 수 있습니다. + 투명도 설정 가능
* 짧은 단어 및 비연속하는 문자 조합으로 검색할 수 있습니다

사용 방법
---
처음 실행하면 아래처럼 실행됩니다.
